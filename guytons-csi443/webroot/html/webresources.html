Web Resources

  • W3Schools
    The W3Schools site contains a comprehensive reference of many web resources.

  • Twitter Bootstrap
    Bootstrap provides an easy to use base on which to start your responsive web site. It uses a mixure of Javascript and CSS to style and organize your site pages. In addition to styling pages, it also allows an easy way to make your site mobile ready with fluid page resizing.

  • Angular
    Your starting point for all things Angular. Angular is a Javascript framework for building responsive websites. It specializes in single page applications by the use of the MVC or MVVM pattern. Perhaps the greatest section of the site for resources is the documentation section that will give the offical documentation on how to use Angular to accomplish your goals.

  • Responsive Web Design by Ethan Marcotte
    The article on the site gets into the hows and whys of responsive web design.

  • Responsive Web Design Basics
    This article by Pete LePage a developer advocate at Google gives you the basics of creating a responsive web site. It also links to a free cours on Udacity to help you actually learn the skills that are being talked about.

  • Code Academy
    This site features interactive training for free for a wide variety of technologies including web design. You can take courses in python, php, sql, Javascript and many others.

  • Free code camp
    Another free site focusing on getting up and running programing and making websites. This site has a strong Javascript focus on web design.
