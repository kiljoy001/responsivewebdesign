Web Resources

  • W3Schools
    The W3Schools site contains a comprehensive reference of many web resources. It has many code samples for html, javascript, sql and other web focused technologies. If you are stuck trying to find some reference material when developing websites, chances are good that you'll be able to find a reference and example here. The comprehensive amount of documentation found here is staggering compared to more focused websites. Indeed, this should be the first stop for syntactical questions or demonstrations on how to use a particular technology.


    Rating: 5 of 5

  • Twitter Bootstrap
    Bootstrap provides an easy to use base on which to start your responsive web site. It uses a mixture of Javascript and CSS to style and organize your site pages. In addition to styling pages, it also allows an easy way to make your site mobile ready with fluid page resizing. Mastering bootstrap will easily make your sites look more professional, as the built in grid structure give much more ease in making your sites look good. Also when using bootstrap you have the option of using themes created by other people to quickly come up with a wining design.


    Rating: 3 of 5

  • Angular
    Your starting point for all things Angular. Angular is a Javascript framework for building responsive websites. It specializes in single page applications by the use of the MVC or MVVM pattern. Perhaps the greatest section of the site for resources is the documentation section that will give the official documentation on how to use Angular to accomplish your goals. With Angular you are able to create complex single page applications without having to create large amounts of html webpages. Additionally with the modular features of Angular, you can easily extend and add features as needed.


    Rating: 3 of 5

  • Responsive Web Design by Ethan Marcotte
    The article on the site gets into the hows and whys of responsive web design as a philosophy with some practical examples on how to perform responsive web design. The author, Ethan Marcotte, argues against static design and instead embracing that in today's world, being flexible is more important than ever as the ever increasing amount of devices with sizes and specifications continue to grow in the marketplace. Within the article he shows some potent examples on how to use CSS to accomplish this philosophy.


    Rating: 2 of 5

  • Responsive Web Design Basics
    This article by Pete LePage a developer advocate at Google gives you the basics of creating a responsive web site. It also links to a free course on Udacity to help you actually learn the skills that are being talked about. By learning from one of the best design originators on the web currently, I strongly feel that you will be able to incorporate the same techniques and technologies to make great webpages as well. Take the course and discover how Google builds responsive websites.


    Rating: 4 of 5

  • Code Academy
    This site features interactive training for free for a wide variety of technologies including web design. You can take courses in python, php, sql, Javascript and many others. If you are just starting out learning how to build websites, this site will help you get started with its interactive exercises and lessons. Although it's not comprehensive, it's good enough to give the basics on how to put together a site and the syntax of what ever language you are trying to build the site in. Give it a try and start learning something new.


    Rating: 5 of 5

  • Free code camp
    Another free site focusing on getting up and running programing and making websites. This site has a strong Javascript focus on web design. As an alternate/supplement to Code Academy, this site also uses a similar style to train you how to develop web sites, however they focus only on Javascript as opposed to have a plethora of different languages and technologies. One of the more interesting features is that there are groups you can find through Facebook or Meetup that will engage in collaborative study


    Rating: 3 of 5
